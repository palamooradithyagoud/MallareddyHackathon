# AI-Powered Intelligent Job Application Assistant (Phase 1)

This project is an AI-powered job application assistant built using a FastAPI backend and a React + TypeScript frontend. It parses resumes and analyzes job descriptions to extract structured data and ATS keywords using Groq API.

## Project Structure
- `frontend/`: React + TypeScript UI application using TailwindCSS and custom styling.
- `backend/`: FastAPI Python web application using SQLAlchemy and Groq services.

## Tech Stack
- **Frontend**: React, TypeScript, TailwindCSS, Axios, React Router.
- **Backend**: FastAPI, SQLAlchemy, SQLite/PostgreSQL, Groq API.
